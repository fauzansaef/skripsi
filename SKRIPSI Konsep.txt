judul : Implementasi Metode Simple Additive Weighting (SAW)
Dalam Menentukan Penugasan Pengerjaan Project pada Direktorat Teknologi Informasi dan Komunikasi DJP

criteria :
- kemampuan coding (newbie, advanced, expert)
- banyaknya project yg sedang dikerjakan
- pengalaman (banyak proyek dan tingkat kesulitan yg pernah dikerjan)
- pelatihan (banyak nya pelatihan yg pernah diikuti)
- penguasaan stack (database, bahasa pemrograman, container, message broker, dll)
note : dalam menentukan nilai pada criteria tsb , menggunakan metode SAW jg 
dengan menghitung detail dari masing2 criteria

- alogritma Simple Additive Weighting (SAW)

role aplikasi : 
- role administrator
- role analis sistem
- role kepala seksi
- role programmer

fitur aplikasi : 
- login -> (nip dan password pegawai) 

- role sistem analis
	-> input project berdasarkan surat ND permintaan aplikasi (project) dari BO (Buisness Owner)
	-> ketika diinput statusnya draft, data yang diinput merupakan hasil analisis dari analis
	-> ada tombol ajukan, untuk diajukan kepada kepala seksi

- role kepala seksi
	-> list persetujuan kemampuan programmer (sesuai criteria)
	-> tombol setujui (ketika diklik muncul modal isinya data profil programmer dan kemampuannya, kemudian dibawah form ada tombol setujui)
	-> list aplikasi yang sudah diajukan kepada kepala seksi
	-> tombol detail (untuk melihat detail requirement project)
	-> tombol tim (menentukan tim yang akan mengerjakan project, hasil yang muncul adalah list data programmer di seksi tsb
	   berdasarkan hasil perhitungan algoritma SAW)
	-> tombol generate surat tugas , status menjadi disetujui

- role programmer
	-> menu input kemampuan programmer (sesuai criteria), ketika diinput data masuk ke kepala seksi untuk disetujui
	-> list aplikasi yang sudah disetujui oleh kepala seksi untuk dikerjakan
	-> tombol lihat surat tugas

- role administrator
	-> input nama pegawai
	-> input kriteria dan nilai (data yang diolah pada algoritma SAW)
	-> upload template surat tugas
	-> input data referensi
